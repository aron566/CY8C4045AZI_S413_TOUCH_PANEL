# CY8C4045AZI_S413_TOUCH_PANEL

## 工程框架

4个实体I/O按键（低电平有效）、1个环形触摸、1个串口、可能的输出脚控制H743电源开关

## 功能

- 四个功能按键：电源、音量增、减、功能选择

  - 电源按键功能：

    - 长按：开关机

    - 单击：锁屏（其他三按键不可用）
    - 双击：解锁（其他三按键可用）

  - 音量按键功能：

    - 单击：增减音量（增至100，减至0）

  - 功能选择按键功能：

    - 单击：切换功能（各功能循环切换）

- 环形触摸按键：在任意环形触摸位置，触摸离开后，位置信息换算成角度设置从机BF算法参数



## 通讯

以主站角色，使用串口以115200波特率与H743通讯

上电自动获取音量数据
